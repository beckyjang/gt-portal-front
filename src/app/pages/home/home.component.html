API 키 가져 오기

API 키를 얻으려면 응용 프로그램을 포털에 추가하십시오. 포털은 등록 된 각 응용 프로그램에 고유 한 API 키를 할당합니다. API에 액세스하려면 API 키가 필요합니다. 또한 API 탐색기에서 API를 테스트하려면 API 키가 있어야합니다.

API 테스트

API 탐색기를 사용하면 거칠거나 wadl 파일이있는 API와 상호 작용할 수 있습니다. 또한 API와 상호 작용하려면 응용 프로그램의 API 키가 필요합니다.

보고서 생성

API의 사용 및 대기 시간을 모니터링하려면 포털의 실시간 분석을 사용하십시오. API 및 애플리케이션의 트래픽 추세를 빠르게 시각화합니다. 조직의 할당량 사용량을 추적합니다. 개발자는 조직 고유의 정보에만 액세스 할 수 있습니다.
